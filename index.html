Case Studies

Real Problems, Real Solutions

Three automation projects I built for real estate and finance operations — each replaced hours of manual work with reliable automated systems.

01Collection Management System

Problem

A real estate firm was tracking collections across multiple spreadsheets. Payment status was unclear, cheque due dates were missed, and monthly collection reports took 3–4 hours to prepare manually.

Solution

Built an automated collection tracking and reporting system that consolidates customer payment data, tracks cheque due dates, and generates management-ready PDF reports — with a Telegram bot sending daily reminders for upcoming cheques.

Tech Stack

  • Python for data processing
  • Excel as data source
  • PDF report generation
  • Telegram Bot API for reminders
  • Cron scheduling for daily runs

My Role

End-to-end: designed the workflow, built the automation scripts, set up the reporting templates, and integrated Telegram for daily alerts.

Business Impact

📉 Reporting time reduced from 3–4 hours to 5 minutes · Zero missed cheque dates since deployment
02Real Estate Booking & CRM System

Problem

A real estate sales team had no centralized system for tracking leads, unit availability, or reservation status. Agents worked from personal notes, customer follow-ups were inconsistent, and management had no visibility into the sales pipeline.

Solution

Designed and built a complete reservation and CRM workflow: lead distribution to agents, real-time unit inventory tracking, reservation status dashboard, payment offer generation, and automated sales-agent commission calculations.

Tech Stack

  • Excel-based data layer for accessibility
  • Python automation scripts
  • PDF payment offer generation
  • HTML/CSS dashboards
  • Integration with existing sales workflows

My Role

Full ownership: requirements gathering, workflow design, implementation, testing with real data, and training the sales team on daily usage.

Business Impact

📋 Unit availability became visible in real time · Reservation processing time cut by ~60% · Commission disputes eliminated
03WhatsApp Reporting & Follow-Up Automation

Problem

Management needed daily operational reports but had to manually compile data from multiple sources. Customer follow-up for payments was done manually via phone calls — time-consuming and inconsistent. Critical information was delayed.

Solution

Built automated workflows that pull data from Excel/ERP sources, process it on schedule, and deliver formatted reports and follow-up messages directly to management and customers via WhatsApp and Telegram — zero manual intervention needed.

Tech Stack

  • n8n for workflow orchestration
  • WhatsApp Business API integration
  • Telegram Bot for internal alerts
  • Python for data processing
  • Scheduled cron jobs for daily execution

My Role

Architected the automation pipeline, configured n8n workflows, integrated messaging APIs, built data processing scripts, and set up scheduling for fully autonomous daily operation.

Business Impact

⏱ 10+ hours/month saved · Reports delivered automatically at 11:30 AM daily · Customer payment response rate improved
